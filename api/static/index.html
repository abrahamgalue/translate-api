API de Traducción

Esta API permite traducir texto de un idioma a otro utilizando el modelo de IA de Google Gemini.

Cómo usar la API

La API tiene un solo endpoint disponible para realizar traducciones entre diferentes idiomas.

GET /api/translate

Este endpoint traduce un texto dado de un idioma a otro.

Parámetros requeridos

  • text: El texto que deseas traducir.
  • from: El idioma de origen (o auto para detectar automáticamente).
  • to: El idioma al que deseas traducir.

Ejemplo de solicitud

GET /api/translate?text=Hola%20mundo&from=Spanish&to=English

Ejemplo de respuesta

{
    "translatedText": "Hello world"
}

Posibles errores

A continuación se muestran los posibles errores que pueden ocurrir al utilizar la API.

Parámetros faltantes

{
    "message": "Missing query params in request",
    "error": true
}

Error interno del servidor

{
    "message": "Internal server error",
    "error": true
}
